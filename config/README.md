# Pasta `config`

Esta pasta será usada futuramente para configurações externas e
explícitas da instituição, como códigos ou parâmetros que não podem ser
inferidos da planilha.

Nesta etapa, caminhos e nomes técnicos ficam centralizados em:

```text
src/config.py
```

Dados regulatórios, identificadores, contas e sistemas não devem ser
incluídos aqui com valores fictícios.
